## 🎯 使用方法

### 方式一：桌面快捷方式（最简单）

1. 双击 `创建桌面快捷方式.bat`
2. 依次双击桌面上的3个快捷方式：
   - `1-安装Mock功能.lnk`
   - `2-启动后端服务.lnk`
   - `3-初始化Mock权限.lnk`
3. 完成！

### 方式二：手动执行

1. **安装数据库**
   ```
   双击：install_mock_feature.bat
   输入：PostgreSQL密码
   等待：看到"安装完成"
   ```

2. **启动后端**
   ```
   双击：start_backend.bat
   等待：看到"Uvicorn running"
   保持窗口打开！
   ```

3. **初始化权限**
   ```
   先登录系统（http://localhost:3000）
   双击：init_mock_permissions.html
   点击："开始初始化权限"
   等待：看到"初始化成功"
   ```

## ❓ 常见问题

### Q: 提示"psql不是内部或外部命令"？
**A**: PostgreSQL未添加到PATH，可以：
- 使用pgAdmin手动执行SQL文件
- 或将PostgreSQL的bin目录添加到系统PATH

### Q: 后端启动失败？
**A**: 检查：
- 8001端口是否被占用
- Python环境是否正确
- 数据库是否运行

### Q: 找不到"Mock数据管理"菜单？
**A**: 
1. 确认前3步都成功
2. 按F5刷新页面
3. 重新登录试试
4. 检查角色权限
