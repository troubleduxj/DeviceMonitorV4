========================================
    Mock功能安装 - 超简单3步走
========================================

📌 重要提示：
   按照以下步骤顺序执行，不要跳步！

========================================
第1步：安装数据库表（1分钟）
========================================

1. 找到文件：
   D:\Cursor\Project\DeviceMonitorV2_20251013_V1\DeviceMonitorV2\scripts\install_mock_feature.bat

2. 双击运行这个文件

3. 看到提示后：
   - 输入 PostgreSQL 的密码（通常是：123456 或 postgres）
   - 按回车键

4. 等待看到：
   ✅ Mock数据表创建成功！
   ✅ Mock管理菜单添加成功！
   🎉 数据库安装完成！

5. 按任意键关闭窗口

========================================
第2步：启动后端服务（1分钟）
========================================

1. 找到文件：
   D:\Cursor\Project\DeviceMonitorV2_20251013_V1\DeviceMonitorV2\scripts\start_backend.bat

2. 双击运行这个文件

3. 等待看到：
   INFO:     Uvicorn running on http://0.0.0.0:8001
   INFO:     Application startup complete.

4. 【重要】看到这个消息后：
   - 不要关闭这个窗口！
   - 让它一直开着运行
   - 这就是后端服务

========================================
第3步：初始化权限（1分钟）
========================================

1. 先登录系统：
   - 打开浏览器
   - 访问：http://localhost:3000
   - 登录系统（使用您的管理员账号）

2. 找到文件：
   D:\Cursor\Project\DeviceMonitorV2_20251013_V1\DeviceMonitorV2\scripts\init_mock_permissions.html

3. 双击用浏览器打开这个文件

4. 点击页面上的"开始初始化权限"按钮

5. 等待看到：
   ✅ 初始化成功！

6. 关闭这个页面

7. 返回系统页面，按 F5 刷新

========================================
✅ 安装完成！开始使用
========================================

1. 刷新系统页面后，在左侧菜单找到：
   "Mock数据管理"

2. 点击进入，就可以开始使用了！

========================================
❓ 常见问题
========================================

Q1: 第1步执行失败，提示"psql不是内部或外部命令"？
A1: 需要安装PostgreSQL或将其添加到系统PATH

Q2: 第2步后端启动失败？
A2: 检查8001端口是否被占用，或查看错误信息

Q3: 第3步提示"未登录"？
A3: 需要先在系统中登录，然后再运行权限初始化页面

Q4: 找不到"Mock数据管理"菜单？
A4: 
   1. 确保前3步都执行成功
   2. 按F5刷新页面
   3. 重新登录试试
