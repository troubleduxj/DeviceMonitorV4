## 💡 **优化方法总结**

### **标准流程**

**步骤 1：添加 TypeScript 支持**
```typescript
<script setup>  →  <script setup lang="ts">
```

**步骤 2：类型化关键 refs**
```typescript
const $table = ref(null)         →  const $table = ref<any>(null)
const queryItems = ref({})       →  const queryItems = ref<Record<string, any>>({})
const pagination = ref({...})    →  const pagination = ref<{page: number; pageSize: number}>({...})
```

**步骤 3：验证零错误**
```bash
npm run lint
```
