## 🏗️ 系统架构

### 技术栈
- **后端**: FastAPI + Python 3.11
- **数据库**: PostgreSQL 15 + Redis 7
- **前端**: Vue 3 + Pinia + Element Plus
- **部署**: Docker + Docker Compose + Nginx
- **监控**: Prometheus + Grafana
- **测试**: Pytest + 集成测试框架

### 架构特点
- **微服务架构**: 模块化设计，易于扩展
- **RBAC权限模型**: 基于角色的访问控制
- **多层缓存**: 本地缓存 + Redis缓存
- **异步处理**: 高性能异步权限验证
- **容器化部署**: Docker容器化，易于部署和扩展
