## 📝 任务更新指南

### 如何更新任务状态
1. **找到对应任务行**
2. **修改状态标记**:
   - ⏳ 待开始 → 🔄 进行中
   - 🔄 进行中 → ✅ 已完成
   - 🔄 进行中 → ❌ 已阻塞
   - ❌ 已阻塞 → 🔄 进行中

3. **填写时间信息**:
   - 实际耗时: 记录真实用时
   - 开始时间: 格式 `2025-01-19 09:00`
   - 完成时间: 格式 `2025-01-19 10:30`

4. **添加备注**: 记录特殊情况或问题

### 状态标记说明
- **⏳ 待开始**: 任务已定义，尚未开始
- **🔄 进行中**: 任务正在开发中
- **✅ 已完成**: 任务已完成并通过测试
- **❌ 已阻塞**: 任务因某些原因无法继续
- **⚠️ 需验证**: 任务完成但需要验证
