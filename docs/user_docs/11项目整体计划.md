# 11 项目整体计划

## Beta V0.1 版本开发计划

**目标：** 完成系统核心框架搭建、基础支撑模块开发、基础功能模块的前后端开发及界面优化，产出可运行的系统原型。

**时间周期：** [待定，例如：4周]

**详细任务列表：**

### 1. 系统整体框架搭建（前后端）

*   **前端 (Vue 3 + TypeScript + Ant Design Vue)：**
    *   初始化 Vue 3 项目，配置 TypeScript 环境。
    *   集成 Ant Design Vue 组件库。
    *   配置 Vue Router 进行路由管理。
    *   配置 Pinia 进行状态管理。
    *   集成 Axios 进行 HTTP 请求封装。
    *   搭建基础页面布局和导航结构。
*   **后端 (FastAPI)：**
    *   初始化 FastAPI 项目，配置项目结构。
    *   集成 Tortoise ORM 或 SQLAlchemy 进行数据库连接和模型定义。
    *   配置 Alembic 或 Aerich 进行数据库迁移管理。
    *   搭建基础认证授权框架（JWT）。
    *   配置日志系统。

### 2. 完善系统支撑基本模块的开发

*   **用户与权限管理：**
    *   后端：实现用户模型、角色模型、权限模型定义。
    *   后端：实现用户注册、登录、登出 API。
    *   后端：实现角色管理（增删改查）API。
    *   后端：实现权限分配（角色与菜单/API关联）API。
    *   前端：开发登录/注册页面。
    *   前端：开发用户管理页面（用户列表、新增/编辑用户、分配角色）。
    *   前端：开发角色管理页面（角色列表、新增/编辑角色、配置权限）。
*   **系统设置：**
    *   后端：实现基础系统参数配置 API。
    *   前端：开发系统设置页面（如数据保留策略、通知方式等）。

### 3. 实现基础功能模块的前后端开发

*   **登录模块：**
    *   前端：实现登录表单验证和提交。
    *   前端：处理 JWT Token 的存储和请求携带。
*   **系统看板：**
    *   后端：提供关键设备指标（在线设备数、告警数）API。
    *   后端：提供实时数据概览 API。
    *   前端：开发系统看板页面，展示概览数据和图表。
*   **设备信息：**
    *   **设备管理页：**
        *   后端：实现设备模型定义。
        *   后端：实现设备增删改查 API。
        *   前端：开发设备列表页面，实现设备展示、搜索、筛选、新增、编辑、删除功能。
        *   前端：实现批量操作功能。
    *   **设备实时监测页：**
        *   后端：提供单个设备实时运行参数 API。
        *   前端：开发设备实时监测页面，集成 ECharts 展示实时曲线图。
    *   **历史数据查询页：**
        *   后端：提供设备历史数据查询 API。
        *   前端：开发历史数据查询页面，实现时间、设备、参数选择，展示数据表格和历史曲线图。
        *   前端：实现数据导出功能。
*   **统计分析：**
    *   **设备在线率统计：**
        *   后端：提供设备在线率统计数据 API。
        *   前端：开发设备在线率统计页面，集成 ECharts 展示统计图表。
        *   前端：实现时间范围选择、设备筛选和数据导出功能。
    *   **焊接时长统计：**
        *   后端：提供焊接时长统计数据 API。
        *   前端：开发焊接时长统计页面，集成 ECharts 展示统计图表。
        *   前端：实现时间范围选择、设备筛选和数据导出功能。

### 4. 系统的界面优化

*   根据 `06前端原型与交互说明.md` 对已开发页面进行 UI/UX 优化。
*   确保响应式设计在不同设备上的良好表现。

## Beta V0.2 版本开发计划

**目标：** 在 V0.1 基础上，逐步实现高级功能模块，提升系统智能化和自动化水平。

**时间周期：** [待定，例如：6周]

**详细任务列表：**

### 1. 告警中心（高级功能）

*   后端：实现告警模型定义、告警规则配置 API。
*   后端：实现告警实时推送机制（如 WebSocket）。
*   前端：开发告警列表页面，实现告警实时展示、处理、确认。
*   前端：开发告警规则配置页面。

### 2. AI 分析与预测

*   后端：集成 AI 模型服务（如 TensorFlow/PyTorch 模型部署）。
*   后端：提供设备健康评分、异常预测、趋势分析 API。
*   前端：开发 AI 分析页面，展示分析结果、报告生成。

### 3. 流程编排

*   后端：设计并实现流程引擎，支持流程定义、执行、监控 API。
*   前端：开发图形化流程编排界面（如基于 GoJS 或 AntV X6），支持拖拽、连接节点。

### 4. 更完善的数据可视化与报表功能

*   后端：提供复杂报表数据生成 API。
*   前端：开发自定义报表功能，支持多维度数据展示和导出。

### 5. 国际化支持

*   前端：集成 Vue I18n，实现多语言资源管理。
*   前端：实现语言切换功能。

### 6. 更全面的系统日志与审计功能

*   后端：完善操作日志记录，支持更细粒度的审计。
*   前端：开发高级审计日志查询页面。

## 持续性任务

*   **代码审查与重构：** 定期进行代码审查，优化代码结构，提升可维护性。
*   **单元测试与集成测试：** 编写和维护测试用例，确保代码质量。
*   **性能优化：** 持续监控系统性能，进行瓶颈分析和优化。
*   **文档更新：** 及时更新各类设计文档和开发手册。
*   **用户反馈与需求迭代：** 持续收集用户反馈，进行需求分析和优先级排序，规划后续版本。
