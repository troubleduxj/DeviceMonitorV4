## ✅ 实施内容

### 1. 设备类型管理页面
**位置**: `web/src/views/device/type/index.vue`

```
操作列新增：[编辑] [配置数据模型 ⭐] [删除]
```

**功能**:
- 点击"配置数据模型"按钮
- 跳转至 `/data-model/config`
- 自动预填充设备类型 (`device_type`, `type_name`)

**用户价值**: 从设备类型管理直达模型配置，无需手动筛选

---

### 2. 设备基础信息页面
**位置**: `web/src/views/device/baseinfo/index.vue`

```
操作列新增：[维修记录] [查看数据 ⭐] [编辑] [删除]
```

**功能**:
- 点击"查看数据"按钮
- 跳转至 `/data-model/preview`
- 自动关联设备信息 (`device_code`, `device_name`, `device_type`)

**用户价值**: 从设备管理直达数据预览，快速查看实时/历史数据

---

### 3. AI监测仪表盘页面
**位置**: `web/src/views/ai-monitor/dashboard/index.vue`

```
页面标题栏：[管理特征模型 ⭐] [刷新数据] [导出报告]
```

**功能**:
- 点击"管理特征模型"按钮
- 跳转至 `/data-model/config`
- 自动筛选AI分析类型 (`model_type: 'ai_analysis'`)

**用户价值**: AI监测依赖特征模型，提供直达管理入口
